# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL querying, interactive dashboards, and business reporting.

---

## Dataset

- Dataset: customer shopping behavior
- Source: [Kaggle / Company / Public Dataset]
- Format: CSV 
- Records: 3900
---

## Tools & Technologies

- Python
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
- SQL / MySQL Server
- Power BI
- Gamma AI (Presentation)
- VS Code

---

## Project Workflow

### 1. Data Loading
- Imported the dataset using Python.
- Examined data structure and column information.

### 2. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Treated inconsistent values.
- Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)
- Performed descriptive statistics.
- Identified trends and patterns.
- Visualized distributions.
- Analyzed relationships between variables.
- Generated business insights.

### 4. SQL Analysis
- Imported cleaned data into SQL/MySQL Server.
- Executed SQL queries to answer business questions.

### 5. Dashboard Development
Created an interactive Power BI dashboard including:
- KPIs
- Charts
- Slicers
- Filters
- Trend Analysis
- Category-wise Performance
- Summary Metrics

### 6. Reporting
- Summarized key findings.
- Generated a business report.
- Created a presentation using Gamma AI.

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/ramadhiny/customer_behavior_analysis
```

### 2. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3. Open the notebook

Run the Jupyter Notebook or open it in VS Code.

### 4. Execute SQL Queries

- Import the cleaned dataset into MySQL Server.
- Run the SQL script provided in the `SQL` folder.

### 5. Open Power BI Dashboard

Open the `.pbix` file using Microsoft Power BI Desktop.
